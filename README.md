# Cheikh's World - Retro OS Portfolio

A personal portfolio website built as a retro operating system experience.

## Features

- **Custom Window Manager**: Draggable, resizable, minimizable windows.
- **Retro Aesthetic**: 90s/2000s inspired design system (Classic Windows/Mac OS vibes).
- **Applications**:
  - **Profile**: Tabbed identity card.
  - **Universe**: Multimedia CD-ROM style interactive experience.
  - **Projects**: IDE/File Manager interface for showcasing work.
  - **Creations**: Photo gallery software.
  - **N2S**: Brand creativity suite (Macromedia/Adobe style).
  - **Graveyard**: Narrative archive of abandoned ideas.
  - **Dino Game**: Retro arcade mini-game.
  - **Instagram**: Social feed terminal.
  - **Computer**: System properties control panel.
  - **Spotify**: Winamp-style music player.

## Tech Stack

- React
- TypeScript
- TailwindCSS
- Lucide React (Icons)

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Run development server:
    ```bash
    npm run dev
    ```
3.  Open http://localhost:5173
